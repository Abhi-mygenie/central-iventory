Placement Rules

WhatsApp icon sits in the left group of the footer, after KOT and Cancel. Same 44×44px touch target. Green WhatsApp brand color (#25D366).
✅
Visible: fOrderStatus 7 (YTC), 1 (Preparing), 2 (Ready), 8 (Hold), 5 (Served), 9 (Pending Payment)
❌
Hidden: fOrderStatus 6 (Paid), 3 (Cancelled), 10 (Reserved)
👆
On tap: Opens a small popup with Customer Name (auto-filled), Phone Number (auto-filled if available), Amount (read-only). Send button triggers the API call.
🔄
Re-tap: Same icon, same popup. Backend handles link reuse (source: "db") vs new link (source: "razorpay").
